### Running this site

1. Clone it
2. `bundle install`
3. `middleman serve`

👍

### Deploying this site

1. `bundle exec rake publish BRANCH_NAME=master`
2. Verify that the [changes are live](http://hey.al) and all is well

_Note: If `bundle exec rake publish BRANCH_NAME=master` fails, `rm -rf build` and try again_

👍👍
